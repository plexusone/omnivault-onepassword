## [v0.3.0] - 2026-05-03

### Highlights

- Renamed to `omni-onepassword` following the omni-<provider> pattern
- Restructured as monorepo with `omnivault/` subdirectory

### Breaking

- Module path changed from `github.com/plexusone/omnivault-onepassword` to `github.com/plexusone/omni-onepassword`
- Import path changed to `github.com/plexusone/omni-onepassword/omnivault`

### Upgrade Guide

- Update import paths from `github.com/plexusone/omnivault-onepassword` to `github.com/plexusone/omni-onepassword/omnivault`
- Run `go get github.com/plexusone/omni-onepassword/omnivault@v0.3.0`

### Changed

- Provider code moved to `omnivault/` subdirectory
- Examples moved to `omnivault/examples/`
- Integration name changed from `omnivault-onepassword` to `omni-onepassword`

### Documentation

- Add root README.md following omni-aws pattern
- Update omnivault/README.md with new import paths

## [v0.2.0] - 2026-03-14

### Highlights

- Module migrated to new GitHub organization: `github.com/plexusone/omnivault-onepassword`
- Updated for 1Password SDK v0.4.0 API breaking changes

### Breaking

- Module path changed from `github.com/agentplexus/omnivault-onepassword` to `github.com/plexusone/omnivault-onepassword` ([`84c7f51`](https://github.com/plexusone/omni-onepassword/commit/84c7f51))

### Upgrade Guide

- Update import paths from `github.com/agentplexus/omnivault-onepassword` to `github.com/plexusone/omnivault-onepassword`
- Run `go get github.com/plexusone/omnivault-onepassword@v0.2.0` to fetch new module

### Changed

- SDK API calls updated from field access to method calls (`client.Items` to `client.Items()`) ([`84c7f51`](https://github.com/plexusone/omni-onepassword/commit/84c7f51))
- List operations changed from iterator pattern to direct slice returns ([`84c7f51`](https://github.com/plexusone/omni-onepassword/commit/84c7f51))

### Dependencies

- Update `github.com/plexusone/omnivault` from v0.2.2 to v0.3.0 ([`84c7f51`](https://github.com/plexusone/omni-onepassword/commit/84c7f51))
- Maintain compatibility with `github.com/1password/onepassword-sdk-go` v0.4.0 ([`84c7f51`](https://github.com/plexusone/omni-onepassword/commit/84c7f51))

### Documentation

- Update README badges and shields for new organization ([`25de397`](https://github.com/plexusone/omni-onepassword/commit/25de397))

### Infrastructure
